# 附录1 - 多项式的带余除法

多项式 $F[x]$（其中系数取值范围 $F$ 是一个域，例如有理数域、实数域或复数域）与整数 $\mathbb{Z}$ 在代数结构上具有高度的相似性。

一、 多项式带余除法的定义
 
:::bg
设 $f(x), g(x)$ 是域 $F$ 上的多项式，且 $g(x) \neq 0$。则存在唯一的多项式 $q(x)$（商式）和 $r(x)$（余式），使得：

$f(x) = q(x)g(x) + r(x)$&nbsp;&nbsp;&nbsp;&nbsp;其中 $r(x) = 0$ 或者 $\deg(r) < \deg(g)$

($\deg$ 表示多项式的次数)
:::

二、证明存在性

:::bg
像整数带余除法一样，不断消掉被除多项式（余式）的最高次项，渐进得到部分商式和新的余式。每次得到的部分商式，等于被除式（上一次除法的余式）的最高次项除以除式的最高次项，也就是把它们的 `系数相除，次数相减`（注：由于要做系数的除法，所以要求系数的取值范围必须构成一个 `数域`，即每一个非 $0$ 系数都有唯一的乘法逆元。这就是为什么谈论多项式时会说 `某某域上的多项式`，指的就是系数的数域）。得到部分商式后，用被除式减去部分商式乘以除式，就得到了这一次消次后的新的余式。

只要新的余式的次数 $\ge$ 除式的次数，就可以继续做除法得到新的部分商式和新的余式，最终必然导致余式为 $0$ 或余式的次数小于除式的次数，就得到了最终的余式 $r(x)$。此时把之前每次得到的部分商式加起来就得到了最终的商式 $q(x)$，这就证明了存在性。
:::
特别说明 - $x$ 的取值范围
:::bg
多项式的系数取值范围必须是一个“数域”，但是多项式的 $x$ 的取值范围，并不要求也是一个数域，因为多项式并不要求对除法封闭（即两个多项式相除不要求也是一个多项式，只有整除的情况才会是）。因此 $x$ 的取值范围只要求是一个 `包含了系数域` 的 `环`。包含系数域是因为任意系数都可以和 $x$ 进行四则运算，“环”是指多项式只需要对乘法和加法封闭。
:::

三、证明唯一性

:::bg
假设存在两组结果 $(q_1(x), r_1(x))$ 和 $(q_2(x), r_2(x))$ 都满足条件，即：
$f(x) = q_1(x)g(x) + r_1(x)$&nbsp;&nbsp;且
$f(x) = q_2(x)g(x) + r_2(x)$
两式相减可得：
$0 = [q_1(x) - q_2(x)]g(x) + [r_1(x) - r_2(x)]$ ，移项得到：
$[q_1(x) - q_2(x)]g(x) = r_2(x) - r_1(x)$
此时我们进行反证：
假设 $q_1(x) \neq q_2(x)$，那么 $q_1(x) - q_2(x) \neq 0$。
根据多项式乘法的次数性质，等式左边的次数为：
$\deg([q_1 - q_2]g) = \deg(q_1 - q_2) + \deg(g) \ge \deg(g)$
然而，对于等式右边，因为 $r_1$ 和 $r_2$ 的次数都严格小于 $g$ 的次数，所以它们的差 $r_2(x) - r_1(x)$ 的次数也必定严格小于 $g$ 的次数（或者为 $0$）：
$\deg(r_2 - r_1) < \deg(g)$
这导致等式左右两边的次数矛盾！因此，假设不成立，必须有 $q_1(x) - q_2(x) = 0$，即 $q_1(x) = q_2(x)$。代回原式即可得到
$r_2(x) - r_1(x) = 0$，即 $r_1(x) = r_2(x)$。
结论：商式和余式是唯一的。
:::
