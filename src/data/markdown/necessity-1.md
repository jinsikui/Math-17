# 猜想和必要性证明（一）

## 4.1 两方面的思路

必要性，就是要找到可以尺规作出的 $n$ 有哪些性质，可以从两方面入手：
- 可以尺规作出的长度有哪些性质
- $\cos(\frac{2\pi}{n})$有哪些相关性质
![两方面思路](necessity_routes.jpg)
两方面重合的部分，也许能建立某种联系或等式，从而找到 $n$ 必须满足的条件。

## 4.2 尺规作图长度构成一个`数域`

因为前面我们已经知道，尺规可作长度集合对 $+-\times\div$ 运算封闭，因此我们请出“数域”理论。
:::bg
**定义4.1（数域）**:
一个“**数域**”，是指一个对 $+-\times\div$ 运算封闭的集合，并且
- 含有加法单位元“0”和乘法单位元“1”
- 任意两元素都可以作 $+-\times$ 运算，任何非“0”元素都可以做除法（能找到乘法逆元），运算结果仍在原集合中（封闭）
- 满足交换律、结合律、分配律
:::
有理数集合 $Q$，实数集合 $R$，复数集合 $Z$，都是数域。
尺规可作的长度集合，也是一个数域，因为允许“开平方”，它比有理数集 $Q$ 大，比实数集 $R$ 小。
尺规作图中，每进行一次“开平方”操作，就有可能发生一次“扩域”。
:::bg
**定义4.2（扩域）**:
向一个数域中，加入一个或多个原数域中不存在的数，进行充分的 $+-\times\div$ 运算后得到新的数域，叫做一次“**扩域**”。
:::
:::bg
例如：
$Q(\sqrt{2})$ —— 向有理数中加入 $\sqrt{2}$ 后得到的数域
$Q(\left\{\sqrt{2}, \sqrt[3]{5}\right\})$ —— 向有理数中加入 $\sqrt{2}$ 和 $\sqrt[3]{5}$ 后得到的数域
$Q(\sqrt{2})(\sqrt[3]{5})$ —— 向有理数中先加入 $\sqrt{2}$ 得到新数域，再加入 $\sqrt[3]{5}$ 得到最终数域
$R(2+3i)$ —— 向实数集中加入复数 $2+3i$ 后得到的数域

“扩域”意味着进行充分的 $+-\times\div$ 运算，例如，向有理数中加入 $\sqrt[3]{5}$，$Q(\sqrt[3]{5})$ 中不仅包含 $\sqrt[3]{5}$，还包含 $5^{\frac{2}{3}} (= \sqrt[3]{5} \times \sqrt[3]{5})$，以及任意有理数 $q$ 乘以 $5^{\frac{2}{3}}$，以及 $\sqrt[3]{5} \div 5^{\frac{2}{3}} (= \frac{5^{\frac{2}{3}}}{5})$，等等。
:::
每次“扩域”，新增的元素很多，有没有规律可循呢？首先需要介绍一些背景知识。
:::bg
**定义4.3（标量）**:
从一个数域中，可以选择任意一个**子数域**，作为原数域的“**标量**”集合。
- 因为标量集也是数域，任何数域都要含有“0”元和“1”元（定义4.1），所以原数域中的“0”元和“1”元肯定在标量集中。
- 一般来说，在“扩域”时，如无特殊说明，就是以之前的域作为标量集合，例如 $Q(\sqrt{2})$、$Q(\sqrt{2}, \sqrt[3]{5})$ 就是以有理数集 $Q$ 作为标量集合，$R(2+3i)$ 是以实数集 $R$ 作为标量集合。
:::
:::bg
**定义4.4（向量和向量空间）**:
一旦选定了标量集合，数域就自动成为一个“**向量空间**”，数域中每一个元素自动成为一个“**向量**”，标量本身也是向量。
:::
:::bg
**定义4.5（线性组合与线性无关）**: 
- **线性组合**：在选定标量集合后，从大数域（向量空间）中任取有限个向量 $v_1, v_2, \dots, v_n$，并从标量集合中任取对应的标量 $c_1, c_2, \dots, c_n$，将它们相乘并相加：$c_1 v_1 + c_2 v_2 + \dots + c_n v_n$，得到的新向量称为这组向量的“**线性组合**”。
- **线性无关**：如果一组向量中，没有任何一个向量可以表示为其他向量的线性组合，这组向量就是“**线性无关**”的。它意味着这组向量“没有任何多余的冗余成分”，每一个都提供了不可替代的新方向。

说明：实际推导中，要证明有限个向量 $v_1, v_2, \dots, v_n$ 之间线性无关，标准的方法就是：任取标量 $c_1, c_2, \dots, c_n$，证明如果 $c_1 v_1 + c_2 v_2 + \dots + c_n v_n = 0$，必有 $c_1 = c_2 = \dots = c_n = 0$
:::
:::bg
**定义4.6（基底）**:
在一个确定了标量集合的数域（向量空间）中，如果能找到一个向量子集合，同时满足以下两个条件，则称这个子集为该空间的一个“**基底**”：
- 生成性（足够用）：大数域中的任意一个向量，都可以表示为该集合中向量的线性组合。
- 线性无关（无冗余）：该集合中的向量相互之间线性无关。

说明：一个向量空间的基底不是唯一的，但是后面我们会证明，基底的“个数”，即“维数”，一定是固定的。
:::
:::bg
**定义4.7（维数）**:
一个数域（向量空间）的某个基底中所包含的向量的个数，定义为该向量空间在此标量集合下的“**维数**”。
- 特别说明：一个数域的“维数”是“相对的”，和所选择的标量集合有关，例如如果选择所研究数域本身作为标量集，那么维数就是1，任何一个非0元都可以作为唯一的基底元素。
:::
:::bg[fold=true, unfoldTitle=特别说明：无穷维情况, foldTitle="收起"]
在更深层的数学中，基底可以有无穷多个，例如以有理数为标量集的实数域就是无穷维的。我们这里只讨论“有限维”情况，即基底只有有限个。（对于无穷维的基底，依然要求每个向量能够被“有限个”基底元素线性表出。无穷个向量间的线性无关定义为“任何一个都不能被有限个其他向量线性表出。但是无穷维空间下，求和和基底的性质证明变得很复杂，本教程也用不到，就不讨论了。后续的讨论都是在“**有限维空间**”这个前提下进行。）
:::
要让以上“基底”的定义有实际意义，我们还要证明：
:::bg
任意（确定了标量集的）数域都可以找到一组线性无关的“基底”，使得任意向量都能被这组基底线性表出。    **(命题4.1)**
:::
:::bg[fold=true, unfoldTitle=展开证明, foldTitle=收起证明]
证明命题4.1（基底的存在性）： 满足定义4.6（既能组合出所有向量，又没有冗余）的“基底”真的存在吗？任意向量真的能被表示出来吗？
我们可以用“贪心策略”来构造它：在大数域中随便拿一个非 0 向量 $e_1$，看它和标量相乘能否覆盖大数域的所有元素。如果能，基底就是 $\{e_1\}$。如果不能，说明大数域里还有“漏网之鱼”。我们把不能被 $e_1$ 线性组合出来的某个向量 $e_2$ 加进来。现在集合变成了 $\{e_1, e_2\}$。它们依然是线性无关的。看 $\{e_1, e_2\}$ 的所有线性组合能否覆盖大数域。如果不能，再找一个“漏网之鱼” $e_3$ 加进来。
不断重复这个过程。因为我们探讨的是有限维空间，这个过程必定会在经过有限次（假设为 $n$ 次）后停止。一旦停止，就意味着再也找不到无法被表示的向量了。此时构造的集合 $\{e_1, \dots, e_n\}$ 就是基底。
:::
:::bg
固定了一组“基底”后，任意向量被这组基底线性表出的方式是唯一的（即向量的坐标唯一）。
**(命题4.2)**
:::
:::bg[fold=true, unfoldTitle=展开证明, foldTitle=收起证明]
证明命题4.2（向量坐标的唯一性）：
假设向量 $V$ 有两种不同的方式被同一组基底 $e_1, \dots, e_n$ 表示：
$$V = a_1 e_1 + \dots + a_n e_n$$
$$V = b_1 e_1 + \dots + b_n e_n$$
两式相减得到：
$$0 = (a_1 - b_1)e_1 + \dots + (a_n - b_n)e_n$$
因为基底是线性无关的，所以括号里的系数必须全为 0。即 
$$a_1 = b_1, \dots, a_n = b_n$$。
这证明了：任意向量不仅能被基底和标量表示，而且其表示的“坐标”（系数）是绝对唯一的。
:::
要上以上“维数”的定义有实际意义，我们要证明：
:::bg
对于不同的基底，基底的个数，即维数，是相同的。    **(命题4.3)**
:::
:::bg[fold=true, unfoldTitle=展开证明, foldTitle=收起证明]
证明命题4.3（维数不变性定理）：
我们要证明的核心逻辑：已知 A 是基底（有 $m$ 个向量），B 也是基底（有 $n$ 个向量）。假设 $m < n$。我们要严格推导出：B 中的 $n$ 个向量必然是线性相关的（从而与 B 是基底的定义产生致命矛盾）。严谨推导步骤：
第一步：写出表示式
设基底 A 的向量为 $e_1, e_2, \dots, e_m$。设基底 B 的向量为 $v_1, v_2, \dots, v_n$。因为 A 是基底（能生成一切），所以 B 中的每一个向量 $v_j$，都可以被唯一地表示为 A 中向量的线性组合。我们把它们写出来：
$$v_1 = a_{11}e_1 + a_{21}e_2 + \dots + a_{m1}e_m$$
$$v_2 = a_{12}e_1 + a_{22}e_2 + \dots + a_{m2}e_m$$
$$\vdots$$
$$v_n = a_{1n}e_1 + a_{2n}e_2 + \dots + a_{mn}e_m$$
（其中所有的 $a$ 都是标量集合里的数）。

第二步：构造线性组合，准备反证
我们要检验 $v_1, \dots, v_n$ 是否线性相关。也就是要试着找一组标量 $x_1, x_2, \dots, x_n$，看看能不能让下式成立，且这组 $x$ 不全为 0：
$$x_1v_1 + x_2v_2 + \dots + x_nv_n = 0$$

第三步：代入并重新整理（纯代数操作）
把第一步里的所有 $v$ 代入到第二步的式子中。展开后，我们按照 $e_1, e_2, \dots, e_m$ 重新把它们合并同类项。整理后的式子长这样：
$$(a_{11}x_1 + a_{12}x_2 + \dots + a_{1n}x_n)e_1 + \dots + (a_{m1}x_1 + a_{m2}x_2 + \dots + a_{mn}x_n)e_m = 0$$

第四步：利用 A 的线性无关性
注意！因为 $e_1, \dots, e_m$ 是基底 A，根据定义，它们本身是线性无关的。几个线性无关的向量加起来等于 0，唯一的可能就是它们前面的系数全部等于 0。所以，上式括号里的每一项都必须是 0，这就自动生成了一组方程：
$$a_{11}x_1 + a_{12}x_2 + \dots + a_{1n}x_n = 0$$
$$\vdots$$
$$a_{m1}x_1 + a_{m2}x_2 + \dots + a_{mn}x_n = 0$$

第五步：得出最终矛盾
仔细看上面这个方程组，它有 $m$ 个方程，但是有 $n$ 个未知数（即 $x_1$ 到 $x_n$）。在我们的反证假设中，$m < n$（方程的个数少于未知数的个数）。
在基础代数中有一个铁律：当齐次线性方程组的未知数个数多于方程个数时，必然存在至少一组不全为 0 的解。 （如果一直消元，最后一定会剩下不受约束的自由变量）。这意味着什么？这意味着我们真真实实地找到了一组不全为 0 的 $x_1, \dots, x_n$，使得 $x_1v_1 + \dots + x_nv_n = 0$ 成立！这就严格证明了 $v_1, \dots, v_n$ 是线性相关的。但这与“B 也是一个基底（必须线性无关）”的定义产生了直接冲突！因此，一开始的假设 $m < n$ 绝对不成立，从而必然有 $m \ge n$。同理，互换 A 和 B 的身份，必然有 $n \ge m$。两者结合，必定得出 $m = n$。
:::

对于前面提到的扩域，我们要证明一条重要的定理：
:::bg
向一个数域 $A$ 中加入一个不在 $A$ 中的数 $\zeta$ 进行扩域得到 $A(\zeta)$（以 $A$ 为标量集合），如果 $\zeta$ 可以表示为某个以 $A$ 中的数为系数的多项式的根，那么其中的**最小多项式**的**次数**（设为 $n$ ）就是 $A(\zeta)$ 维数，并且 $1, \zeta, \zeta^2, \dots, \zeta^{n-1}$ 就是一组基。    **(命题4.4)**

注：以 $\zeta$ 为根的“多项式”，是指满足方程 $p(\zeta) = a_{n}\zeta^n + a_{n-1}\zeta^{n-1} + \dots + a_1\zeta + a_0 = 0$
（$a_{n}, a_{n-1}, \dots , a_0 \in A, a_{n} \neq 0$）的多项式 $p(x)$。最高次项的次数 $n$ 称为这个多项式的“次数”，其中一定能找到一个 $n$ 最小的多项式，称为以 $\zeta$ 为根的“最小多项式”。
:::
:::bg[fold=true, unfoldTitle=展开证明, foldTitle=收起证明]
第一部分：证明 $1, \zeta, \zeta^2, \dots, \zeta^{n-1}$ 是“线性无关”的
核心逻辑：反证法推出和 $n$ 是最小次数矛盾。
反证法。假设这组元素是线性相关的。根据定义，这意味着存在一组在标量域 $A$ 中不全为 0 的系数 $c_0, c_1, \dots, c_{n-1}$，使得：
$$c_0 \cdot 1 + c_1 \zeta + c_2 \zeta^2 + \dots + c_{n-1} \zeta^{n-1} = 0$$
我们把这组系数提出来，构造一个新的多项式 $f(x)$：
$$f(x) = c_0 + c_1 x + c_2 x^2 + \dots + c_{n-1} x^{n-1}$$
根据第 1 步的假设，把 $\zeta$ 代入 $f(x)$，我们得到 $f(\zeta) = 0$。这意味着 $\zeta$ 是多项式 $f(x)$ 的一个根。致命矛盾出现： $f(x)$ 的最高次数最多只有 $n-1$。但我们在前提里已经规定了，以 $\zeta$ 为根的“最小多项式”的次数是 $n$！一个次数小于 $n$ 的非零多项式绝不可能以 $\zeta$ 为根。
结论： 唯一的可能，就是 $f(x)$ 必须是一个“零多项式”。即所有的系数 $c_0 = c_1 = \dots = c_{n-1} = 0$。这就严格证明了 $1, \zeta, \dots, \zeta^{n-1}$ 之间没有任何非平凡的线性抵消，它们绝对是线性无关的。

第二部分：证明它们能“生成”扩域 $A(\zeta)$ 中的任何元素（生成性）
核心逻辑：这是整个证明最精彩的部分。我们需要证明，无论对 $\zeta$ 进行多么复杂的加减乘除，最终都可以“降维打击”成次数不超过 $n-1$ 的多项式。
由于 $A(\zeta)$ 是一个对 $+ - \times \div$ 封闭的数域，它里面的任意一个元素，一定是经过有限次运算得到的。根据数学归纳法，我们只需要证明，每一步运算的结果都可以被我们的基底线性表出。我们只需要证明其中“最复杂”的一步运算，即两个关于 $\zeta$ 的多项式相除的形式：$\frac{f(\zeta)}{g(\zeta)}（其中 g(\zeta) \neq 0）$ 能被基底线性表出即可。我们分两步走：

步骤 2.1：处理加减乘（降次）
假设只有一个多项式 $f(\zeta)$。无论它的次数有多高（比如 $\zeta^{100}$），我们怎么把它塞进 $1 \sim \zeta^{n-1}$ 的框架里？
对普通的未知数 $x$，我们使用初等代数的多项式带余除法（见附录1），用 $f(x)$ 除以最小多项式 $p(x)$（写成乘加的形式）：
$$f(x) = q(x)p(x) + r(x)$$
（其中商是 $q(x)$，余数是 $r(x)$。根据除法原理，余数 $r(x)$ 的次数必定严格小于除数 $p(x)$ 的次数 $n$）。我们把 $x = \zeta$ 代入上式：
$$f(\zeta) = q(\zeta)p(\zeta) + r(\zeta)$$
因为 $p(\zeta) = 0$，所以中间那一项直接归零！
$$f(\zeta) = q(\zeta) \cdot 0 + r(\zeta) = r(\zeta)$$
结论： 任何关于 $\zeta$ 的高次多项式 $f(\zeta)$，在数值上都等于一个次数小于 $n$ 的多项式 $r(\zeta)$。而 $r(\zeta)$ 显然是 $1, \zeta, \dots, \zeta^{n-1}$ 的线性组合！

步骤 2.2：处理除法（化除为乘）
这是初学者最难跨过的一道坎：如果有个分母 $g(\zeta)$ 怎么办？分式怎么能变成多项式呢？
因为 $g(\zeta) \neq 0$，说明 $\zeta$ 不是 $g(x)$ 的根。这意味着最小多项式 $p(x)$ 不能整除 $g(x)$（注意这里说的是多项式的除法不是数的除法）。
既然 $p(x)$ 是满足条件的最小多项式，它在标量域 $A$ 上是不可约的（不能分解成两个更低次多项式的乘积，否则 $\zeta$ 就会是其中一个更低次多项式的根）。
因为 $p(x)$ 不可约，且不整除 $g(x)$，所以它们两者的最大公因式必定是 1。根据多项式版本的裴蜀定理（多项式辗转相除法，见附录2），既然最大公因式是 1，就必定存在两个多项式 $u(x)$ 和 $v(x)$（系数都在 $A$ 中），使得：
$$u(x)g(x) + v(x)p(x) = 1$$
见证奇迹的时刻 —— 把 $x = \zeta$ 代入上式：
$$u(\zeta)g(\zeta) + v(\zeta) \cdot 0 = 1$$
$$u(\zeta)g(\zeta) = 1$$
$$\frac{1}{g(\zeta)} = u(\zeta)$$
结论： 在这个扩域里，除以 $g(\zeta)$，其实在代数上完全等价于乘以另一个多项式 $u(\zeta)$！ 根本没有真正的“分式”。

第二部分总爆发：任意一个元素 $\frac{f(\zeta)}{g(\zeta)}$，等于 $f(\zeta) \times u(\zeta)$。两个多项式相乘，结果还是一个多项式。再根据步骤 2.1，任何多项式都可以通过除以 $p(x)$ 取余数，降次为一个次数不超过 $n-1$ 的多项式。这就严格证明了：$A(\zeta)$ 中的任何元素，都可以表示为 $1, \zeta, \zeta^2, \dots, \zeta^{n-1}$ 加上 $A$ 中系数的线性组合！

最终结论：集合 $\{1, \zeta, \zeta^2, \dots, \zeta^{n-1}\}$ 既是线性无关的（第一部分），又能生成全空间（第二部分）。因此，它完美满足了之前给出的定义 4.6，必定是向量空间 $A(\zeta)$ 的一组基底。因为这组基底里面恰好有 $n$ 个元素，根据定义 4.7，$A(\zeta)$ 在标量域 $A$ 上的维数就是 $n$。

Q.E.D. (证明完毕)
:::

对于尺规作图，每次产生一个新的长度 $\zeta$，如果这个长度不在之前长度生成的数域中，那这个新长度一定是“开平方”得到的（因为数域对加减乘除封闭），也就是说：
:::bg
$\zeta = \sqrt{a}, a \in A$ $A$ 是之前已经得到的长度生成的数域。
:::
将上面的式子变形一下：
:::bg
$\zeta^2 - a = 0$
这意味着 $\zeta$ 是多项式 $p(x) = x^2 - a$ 的根，那么 $p(x)$ 是不是以 $\zeta$ 为根的最小多项式呢？
$p(x)$ 的次数是 $2$ 已经很小了，如果次数更小只能是 $1$，那意味着存在多项式 $p'(x) = x - b$, $b \in A$ 以 $\zeta$ 为根，也就是 $\zeta - b = 0$, $\zeta = b$，这意味着新的长度 $\zeta$ 还在原来的数域 $A$ 中，并没有发生扩域（这种情形例如由原来的长度 $4$ 开方得到长度 $2$ 并没有扩域）。因此，如果新的长度 $\zeta$ 确实导致了扩域（比如由原来的长度 $5$ 开方得到长度 $\sqrt{5}$)，那么以 $\zeta$ 为根的最小多项式的次数一定只能是 $2$。由命题4.4，$A(\zeta)$ 的维数一定是 $2$ (以产生 $\zeta$ 前的数域 $A$ 作为标量集)。总结一下有下面命题：
:::
:::bg
尺规作图每次开平方得到一个新长度，如果这个长度不在之前长度生成的数域中 (即发生了“扩域”)，那么新数域的维数 (以之前长度生成的数域为标量集)，一定是2。  **(命题4.5)**
:::
上面我们证明了，$A(\zeta)$ 的维数是 $2$，这是以中间某一步达到的数域 $A$ 作为标量集的。如果我们以最初的数域 $Q$（有理数集）作为标量集，那么 $A(\zeta)$ 的维数是多少呢？这就要用到下面的两个定理：
:::bg
$A(X \cup Y) = A(X)(Y)$  **(命题4.6)**
即扩域结果和扩域的顺序无关。
你可以一次性把所有新元素倒进去，也可以分批一点点加，结果完全一样。
:::
:::bg[fold=true, unfoldTitle=展开证明, foldTitle=收起证明]
求证：$A(X \cup Y) = A(X)(Y)$
核心逻辑： 在代数中，对于“由某集合生成的数域”，最严谨的定义是：$A(S)$ 是所有既包含数域 $A$ 又包含集合 $S$ 的数域中，最小的那一个。要证明两个集合相等，最标准的做法是证明它们互相包含（即证明左边 $\subseteq$ 右边，且右边 $\subseteq$ 左边）。

第一步：证明 $A(X \cup Y) \subseteq A(X)(Y)$
观察右边的集合 $A(X)(Y)$。根据定义，它是以 $A(X)$ 为基础，加入 $Y$ 后生成的数域。因此，它必定包含 $A(X)$，也包含集合 $Y$。既然它包含数域 $A(X)$，而 $A(X)$ 又包含了 $A$ 和 $X$，那么 $A(X)(Y)$ 实际上同时包含了数域 $A$、集合 $X$ 以及集合 $Y$。换句话说，$A(X)(Y)$ 是一个既包含数域 $A$，又包含集合 $X \cup Y$ 的数域。而左边的 $A(X \cup Y)$，根据定义，是所有既包含 $A$ 又包含 $X \cup Y$ 的数域中最小的一个。既然右边满足条件，而左边是最小的，那么左边必定被包含在右边之内。即：$A(X \cup Y) \subseteq A(X)(Y)$。

第二步：证明 $A(X)(Y) \subseteq A(X \cup Y)$
首先根据定义，显然有 $A(X) \subseteq A(X \cup Y)$。现在我们盘点一下 $A(X \cup Y)$ 拥有的元素：它包含了数域 $A(X)$；包含了集合 $Y$。即：$A(X \cup Y)$ 是一个既包含了数域 $A(X)$，又包含了集合 $Y$ 的数域。接下来，我们看目标集合 $A(X)(Y)$ 的定义。它被定义为所有既包含数域 $A(X)$，又包含集合 $Y$ 的数域中最小的一个。最终推导： 既然 $A(X \cup Y)$ 是一个符合条件（包含 $A(X)$ 和 $Y$）的数域，而 $A(X)(Y)$ 是符合该条件的最小数域，那么这个最小的数域必定被包含在前者之中。因此严格得出：$A(X)(Y) \subseteq A(X \cup Y)$。

结论：
既然相互包含，两边必定完全相等：$A(X \cup Y) = A(X)(Y)$。这就严谨地证明了扩域可以分步进行。
:::
关于域的维数的“塔定理”：
:::bg
定义维数函数 $d(D, S)$ 为以 $S$ 为标量域的数域 $D$ 的维数 ($S \subseteq D$)。
设 $A$ 是一个数域，$X$ 和 $Y$ 是用来扩域的集合，那么我们断言：
$$d(A(X \cup Y), A) = d(A(X), A) \times d(A(X \cup Y), A(X))$$
如果令 $K = A(X)$，也可以写成：
$$d(A(X \cup Y), A) = d(A(X), A) \times d(K(Y), K)$$
**(命题4.7)**
:::
:::bg[fold=true, unfoldTitle=展开证明, foldTitle=收起证明]
求证：在大域 $A(X \cup Y)$ 上的维数，等于中间域 $A(X)$ 在 $A$ 上的维数，乘以大域在中间域上的维数。
为了表述清晰且绝对严谨，我们先统一下符号。
设基础域为 $A$。
设中间域为 $K = A(X)$。
设最高域为 $L = A(X \cup Y)$，根据命题1，它也等于 $K(Y)$。
设 $K$ 作为 $A$ 上的向量空间，维数为 $m$。
设 $L$ 作为 $K$ 上的向量空间，维数为 $n$。
我们需要严格证明：$L$ 作为 $A$ 上的向量空间，维数恰好是 $m \times n$。

核心逻辑：用两组小基底，人为构造出一组大基底，并证明它完美满足“基底”的两个硬性条件（生成性、线性无关）。
准备工作：
因为 $K$ 在 $A$ 上的维数是 $m$，所以它必然有一组基底，设为 $e_1, e_2, \dots, e_m$。
因为 $L$ 在 $K$ 上的维数是 $n$，所以它也必然有一组基底，设为 $f_1, f_2, \dots, f_n$。
我们把这两组基底两两相乘，构造出一个包含 $m \times n$ 个元素的新集合 $S$：
$$S = \{e_i f_j \mid 1 \le i \le m, 1 \le j \le n\}$$
接下来，我们分两步严格证明 $S$ 就是最高域 $L$ 在基础域 $A$ 上的基底。

第一步：证明 $S$ 具有生成性（能表示 $L$ 中的任意元素）
1. 从最高域 $L$ 中任取一个元素 $v$。
2. 因为 $f_1, \dots, f_n$ 是 $L$ 在 $K$ 上的基底，所以 $v$ 必然可以写成它们的线性组合（标量取自 $K$）：
$$v = c_1f_1 + c_2f_2 + \dots + c_nf_n \quad (其中 c_1, \dots, c_n \in K)$$
3. 仔细看这些系数 $c$。因为它们都在中间域 $K$ 里，而 $e_1, \dots, e_m$ 是 $K$ 在 $A$ 上的基底，所以每一个 $c_j$ 又可以写成 $e_1, \dots, e_m$ 的线性组合（标量取自 $A$）：
$$c_j = a_{1j}e_1 + a_{2j}e_2 + \dots + a_{mj}e_m \quad (其中所有的 a \in A)$$
4. 我们把第 3 步的式子代回到第 2 步中，用分配律展开：
$$v = \sum_{j=1}^n c_j f_j = \sum_{j=1}^n \left( \sum_{i=1}^m a_{ij} e_i \right) f_j = \sum_{j=1}^n \sum_{i=1}^m a_{ij} (e_i f_j)$$
5. 看这个最终结果！ $a_{ij}$ 是最底层数域 $A$ 中的元素，而 $e_i f_j$ 正是我们构造的集合 $S$ 中的元素。这就证明了，$L$ 中的任意元素 $v$，全都可以被集合 $S$ 用 $A$ 中的标量线性组合出来！生成性得证。

第二步：证明 $S$ 是线性无关的（无冗余）
这是反证法的天下。假设 $S$ 中的元素线性相关，即存在一组在基础域 $A$ 中不全为 0 的系数 $a_{ij}$，使得：
$$\sum_{j=1}^n \sum_{i=1}^m a_{ij} (e_i f_j) = 0$$
1. 我们把式子稍微变形，把关于 $i$ 的求和用括号括起来，把 $f_j$ 提出来：
$$\sum_{j=1}^n \left( \sum_{i=1}^m a_{ij} e_i \right) f_j = 0$$
2. 注意看括号里的这一坨 $\left( \sum_{i=1}^m a_{ij} e_i \right)$。因为 $a_{ij}$ 都在 $A$ 里，$e_i$ 都在 $K$ 里，所以加起来的这一坨，它是一个中间域 $K$ 里的元素。我们把它记作 $C_j$。
3. 那么式子变成了：
$$C_1f_1 + C_2f_2 + \dots + C_nf_n = 0 \quad (其中 C_j \in K)$$
4. 现在，调用基底的铁律！因为 $f_1, \dots, f_n$ 是 $L$ 在 $K$ 上的基底，它们在标量域 $K$ 上必须是线性无关的！一堆线性无关的向量加起来等于 0，唯一的可能就是系数全部为 0。
5. 因此，必须有：$C_1 = 0, C_2 = 0, \dots, C_n = 0$。
6. 我们再把 $C_j$ 展开看：
$$a_{1j}e_1 + a_{2j}e_2 + \dots + a_{mj}e_m = 0$$
7. 再次调用基底的铁律！因为 $e_1, \dots, e_m$ 是 $K$ 在 $A$ 上的基底，它们在标量域 $A$ 上也是线性无关的！
8. 因此，所有的系数 $a_{1j}, a_{2j}, \dots, a_{mj}$ 必须全部为 0。
9. 再根据 $j$ 任意性，我们得到所有的 $a_{ij}$ 都必须全部为 0.

产生矛盾： 我们在开头假设了这组系数“不全为 0”，但经过严格推导，它们必须“全部为 0”。假设不成立。因此，集合 $S$ 绝对是线性无关的。
最终结论：我们人为构造的集合 $S$，既能生成全空间，又线性无关，它就是最高域 $L$ 在最底层基础域 $A$ 上的基底。而 $S$ 里面包含元素的个数恰好是 $m \times n$。所以，$d(A(X \cup Y),A) = d(A(X),A) \times d(K(Y),K)$。证毕。
:::
我门再回到尺规作图，由于尺规作图每次开平方，如果发生“扩域”，新数域相对扩域前的维数都是 2 (命题4.5)，那么再根据命题4.7，尺规作图在固定有限次开平方后，能够做出的长度的集合，相对于最初的有理数集 $Q$ 的维数，一定是 2 的次方的形式，即：
:::bg
尺规作图在固定 $p$ 次开平方后，能够做出的长度集合，它的维数 (以有理数集 $Q$ 为标量)，一定能表示为 $2^k$ ($k$ 是 $\ge 0$ 且 $\le p$ 的整数，等于开平方导致“扩域”的次数)。  **(命题4.8)**
:::
:::bg
实际举例：
假设尺规作图第一次开平方得到的“无理数”是 $\sqrt{5}$
$Q(\sqrt{5})$ 相对于 $Q$ 的维数是 $2$，基底是 $\{1,\sqrt{5}\}$
然后后面某步又对 $\sqrt{5}$ 进行了开平方得到 $\sqrt[4]{5}$
$Q(\left\{\sqrt{5}, \sqrt[4]{5}\right\})$ 相对于 $Q(\sqrt{5})$ 的维数是 $2$，基底是 $\{1, \sqrt[4]{5}\}$
$Q(\left\{\sqrt{5}, \sqrt[4]{5}\right\})$ 相对于 $Q$ 的维数是 $2 \times 2 = 4$，基底是:
$$\{1,\sqrt{5}\} \cdot \{1, \sqrt[4]{5}\} = \{1 \times 1, 1 \times \sqrt{5}, \sqrt[4]{5} \times 1, \sqrt[4]{5} \times \sqrt{5}\} = \{1, \sqrt{5}, \sqrt[4]{5}, 5^{\frac{3}{4}}\}$$
后面某步又对 $7$ 开平方得到 $\sqrt{7}$
$Q(\left\{\sqrt{5}, \sqrt[4]{5}, \sqrt{7}\right\})$ 相对于 $Q$ 的维数是 $2 \times 2 \times 2 = 8$，基底是:
$$\{1, \sqrt{5}\} \cdot \{1, \sqrt[4]{5}\} \cdot \{1, \sqrt{7}\} = \{1, \sqrt{5}, \sqrt[4]{5}, 5^{\frac{3}{4}}, \sqrt{7}, \sqrt{35}, \sqrt{7}\sqrt[4]{5}, \sqrt{7}\cdot5^{\frac{3}{4}}\}$$
后面某步又对 $28$ 开平方得到 $2\sqrt{7}$, 由于未发生扩域，维数和基底不变。
后面如果不再开平方，或即使开平方但不再扩域，所有尺规可作长度的集合就是 $\{1, \sqrt{5}, \sqrt[4]{5}, 5^{\frac{3}{4}}, \sqrt{7}, \sqrt{35}, \sqrt{7}\sqrt[4]{5}, \sqrt{7}\cdot5^{\frac{3}{4}}\}$ 和有理数集 $Q$ 的线性组合能得到的长度。
